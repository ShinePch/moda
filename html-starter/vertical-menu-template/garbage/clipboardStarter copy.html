Parsec 소개
  • 초저지연 원격 데스크톱
  • 4K 60fps 게임 스트리밍
  • 간편한 설정과 사용
  • 암호화된 안전한 연결
시스템 요구사항
  • Windows 10/11 (64비트)
  • DirectX 11 지원 그래픽카드
  • 안정적인 인터넷 연결
  • 최소 4GB RAM 권장
설정 가이드
#설정 #가이드

설치 순서
  1. 위의 다운로드 버튼으로 Parsec 설치파일 다운로드
  2. 다운로드된 parsec-windows.exe 파일 실행
  3. 설치 마법사 따라 설치 진행
  4. Parsec 계정 생성 또는 로그인
  5. 호스트 설정 또는 게스트 연결 설정

공식 사이트: https://parsec.app

호스트 설정 단계
  1. Parsec 실행 후 로그인
  2. 설정(Settings) → 호스트(Host) 탭 이동
  3. '호스트 활성화' 체크박스 활성화
  4. 해상도, 대역폭, 품질 설정 조정
  5. 친구 또는 팀원에게 컴퓨터 ID 공유
주의: 방화벽에서 Parsec 허용 필요

게스트 연결 방법
  1. Parsec 실행 후 로그인
  2. 메인 화면에서 '컴퓨터 추가' 클릭
  3. 호스트로부터 받은 컴퓨터 ID 입력
  4. 연결 요청 승인 대기
  5. 연결 완료 후 원격 데스크톱 사용
팁: 게임 모드에서는 마우스와 키보드가 자동으로 최적화됩니다
